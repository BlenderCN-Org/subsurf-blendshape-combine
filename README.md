# subsurf-blendshape-combine

Subsurfを有効にした状態でブレンドシェイプを作ってExportする際に困ったことはないでしょうか？
そんな時は、このスクリプトを使ってブレンドシェイプを維持したままSubsurfを適応しましょう。
